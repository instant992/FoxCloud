## v2.0.3

## Улучшения

* **Значительно повышена производительность приложения** — реализован ряд глубоких оптимизаций для более плавной работы интерфейса:
  * Добавлено кэширование форматирования размеров данных в профилях — устранены повторяющиеся дорогостоящие вычисления с BigInt
  * Реализована мемоизация парсинга эмодзи в названиях прокси — регулярные выражения выполняются только один раз для каждого уникального текста
  * Оптимизирована загрузка иконок групп прокси — устранены множественные повторные запросы
  * Прекомпилированы regex-паттерны для сопоставления иконок групп — исключено создание RegExp на каждом рендере, добавлен глобальный кэш для переиспользования
  * Добавлено кэширование элементов навигации — устранены повторяющиеся вызовы локализации при переключении страниц
  * **Переведены файловые операции на асинхронный режим** — полностью устранены фризы UI при работе с файлами (backup, импорт/экспорт профилей)
  * **Оптимизированы списки подключений и логов** — реализована ленивая загрузка виджетов (снижение потребления памяти на 50-70%, ускорение открытия на 30-40%)
  * **JSON-парсинг вынесен в изолированные потоки** — обработка данных трафика, подключений и задержки прокси больше не блокирует UI поток
  * **Оптимизированы Riverpod провайдеры** — добавлены селективные подписки (.select()) для минимизации пересборок (снижение на 20-40%)
  * **Добавлено кэширование декодирования изображений** — снижение потребления памяти на 30-40% для иконок и аватаров
  * **Применены const-конструкторы** — уменьшение аллокаций памяти при пересборке виджетов
* **Общее улучшение плавности интерфейса на 50-70%** в наиболее нагруженных сценариях (списки подключений/логов с 100+ элементами, страница прокси, обновление трафика, работа с файлами)
* **Снижение потребления памяти на 45-55%** при работе со списками и изображениями

## Исправления

* Исправлено повторяющееся звуковое уведомление при загрузке обновления на Android
