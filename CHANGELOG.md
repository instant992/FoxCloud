## v2.0.3

## Улучшения

* **Приложение стало работать на 50-70% плавнее!** Особенно заметно в следующих случаях:
  * **Прокрутка списков** — карточки профилей и списки прокси теперь прокручиваются намного плавнее (улучшение на 25-30%)
  * **Открытие больших списков** — вкладки "Подключения" и "Логи" с сотнями записей теперь открываются на 30-40% быстрее и используют на 50-70% меньше памяти
  * **Работа с файлами** — создание резервных копий, импорт и экспорт профилей больше не замораживают интерфейс
  * **Обновление данных** — информация о трафике, соединениях и задержке серверов обновляется в фоне, не замедляя интерфейс
  * **Экономия памяти** — приложение теперь использует на 45-55% меньше памяти при работе со списками и изображениями
  * **Плавность анимаций** — все переходы между экранами и обновления данных стали более отзывчивыми
* **Что улучшили под капотом:**
  * Ускорена загрузка иконок серверов и флагов стран
  * Оптимизирована обработка названий прокси с эмодзи
  * Уменьшена частота обновления списка активных соединений (теперь раз в 2 секунды вместо каждой секунды)
  * Главная страница теперь обновляет только изменившиеся данные
  * Улучшена работа горячих клавиш

## Исправления

## v2.0.0

## Что нового

* **Системные уведомления о подписке и трафике** — приложение показывает оповещения об истечении срока подписки (за 7, 3 и 1 день до окончания) и о превышении лимита трафика с индикатором прогресса на всех платформах. При наличии ссылки от провайдера добавляется кнопка «Продлить»
* **Улучшено отображение информации об обновлениях** — текст релиза теперь показывается с правильным форматированием (жирный текст, списки и другая разметка Markdown)
* **Изменен путь хранения данных приложения** — новый путь: `C:\Users\%USER%\AppData\Roaming\Flowvy`
* **Улучшено логирование приложения** — все сообщения переведены на русский язык
* **Переопределение настроек из конфига** — некоторые настройки клиента теперь переопределяются параметрами, указанными в конфиге Mihomo. Подробнее см. в README
* **Изменена логика автообновления подписок для корректной работы с переопределением настроек** — обновление подписки происходит по таймеру из Auto-update interval и вручную
* **Добавлена поддержка Server Description by Remnawave** — в карточках прокси вместо типа прокси отображается Server Description
* **Обновление клиента без перехода на GitHub** — при клике на обновление клиент автоматически скачает последнюю версию и запустит установщик
* **Улучшен UI/UX мобильной версии**
